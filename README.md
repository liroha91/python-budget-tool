# python-budget-tool
A beginner-to-intermediate Python budgeting application designed to track:
- income
- bills
- debt
- savings goals
- transactions
- account balances

Features:
- Add and manage transactions
- Calculate total income and expenses
- Track recurring bills
- Monitor debt payoff progress
- Track savings goals
- View leftover monthly spending money

Technologies used:
- Python
- Object-oriented Programming
- File handling/CSV
- Git + GitHub

Future goals:
- CSV importing
- SQLite database support
- GUI interface
- Charts/graphs
- Web version
